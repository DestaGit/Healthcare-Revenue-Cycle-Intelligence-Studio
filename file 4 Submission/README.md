# RevenueCycle Intelligence Studio

## Features

-   Standalone browser application
-   No backend
-   Embedded synthetic dataset
-   CSV upload
-   Revenue leakage analytics
-   Denial intelligence
-   KPI scorecards
-   Executive dashboard
-   Report generation
-   Local browser storage

## Run

Open `index.html` in any modern browser.

-   CSS3
-   Vanilla JavaScript
